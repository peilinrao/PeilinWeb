GST603

GST603 is an online chatroom developed by Peilin Rao and Jiyu Hu. It is named by the name of the apartment which the developers live in. It supports multithread group chat, user interface, uploading and downloading file, and voice message.

VR Running

VR Running is a game developed by Peilin Rao, Yuan Zhang, Yu Chen and Boom Dej-Udom. It is inspired by the famous temple run and provides fantastic VR experiment. The play can switch among three weapons to destroy different obstacles in order to run further.

MentOS

MentOS is a simplified linux-like system developed by Peilin Rao, Jiyu Hu, Zaizhou Wu and Ribhav Jain. It supports thread-safe handling for exceptions, system calls and device interrupts. Also we implemented fully functional paging, task scheduling and file system.

Gomoku

A Gomoku game with Monte Carlo Tree Search AI that also supports local network multiplaying.

Audio Classifier

A speaker classification program implemented using Long Short Term Memory of recurrent neural network architecture. It takes in WAV file for 20 different English speaker. It can tell the confidence of which person is speaking given a piece of audio.
